# BannerFlow Manager

Веб-приложение для управления рекламными баннерами в играх.

## Структура проекта
- `bannerflow_project/` - настройки Django
- `campaigns/` - приложение для рекламных кампаний
- `users/` - приложение для пользователей
- `analytics/` - приложение для аналитики

## Модели
- Game (игра)
- Campaign (рекламная кампания) 
- Banner (баннер)

## Технологии
- Python 3.13
- Django 5.2.8
- SQLite
